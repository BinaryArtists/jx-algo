### 二叉树的高度/深度

https://www.jianshu.com/p/a424af3540e0

* 广度优先搜索
算法思想：用队列实现层序遍历，返回最后一层的高度。
* 深度优先搜索 1
算法思想：递归求左右子树高度，较大者 +1。
* 深度优先搜索 2
算法思想：用栈实现前序（或后序）遍历，最大栈长度即为树的高度。
